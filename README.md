# Number Guessing Game

Small C# console game to practice basics of C# / .NET and programming.

Quick start
-----------
- Requirements: .NET 10 SDK
- Run: `dotnet run` (from the project folder)

How to play
-----------
The program picks a random number. Enter guesses in the console and the game tells you if the secret number is higher or lower until you win.
